Aquecimento, exercício 0.1: Entendendo a saída de testes

Nesta atividade de JavaScript, vamos apresentar os resultados de cada exercício usando uma função criada pelos professores chamada escreva(...). Ela mostra na página web o valor de uma determinada variável, com um determinado rótulo (descrição) para identificá-la. Assim, essa função recebe como parâmetros: 1) o número do exercício, 2) o rótulo (que é uma descrição da variável) e 3) o valor da variável. Por exemplo, ela pode ser invocada assim: 

function algumaFuncaoMinha() {
    // ...
    // ...
    // ...
    let senhaWiFiDoCoutinho = 'votei17fizcagada';
    // ...
    // ...
    // chamando função 'escreva(...)'
    // ^
    // -------------------------------------------------------------------
    escreva(1, 'A senha do WiFi na casa do Coutinho', senhaWiFiDoCoutinho);
    //      -  -------------------------------------  --------------------
    //      ^ nº exercício   ^ rótulo (descrição)     ^ variável
    // ...
    // ...
}

Veja, no arquivo exercicios.js, na função testeSaidas, como a função escreva(...) foi chamada para imprimir a variável a. Ao imprimir, ela aparecerá na caixa de saída (logo abaixo). Note que a função está apenas implementada mas ainda não foi invocada (chamada). Você pode invocá-la no próprio arquivo (exercicio.js). Assim que chamar, você verá o resultado da variável na caixa de saída logo abaixo. Lembre-se também que, para verificar se ocorreu algum erro de sintaxe (escrita) no seu código, pressione F12 e o erro aparecerá na aba de console.

Exercício 1: Altura da Bola no Instante t

Exercicio adaptado de [Fundamentos da Programação de Computadores: Algoritmos, Pascal, C, C++ e Java, Holloway, 2008]

Uma bola é jogada para o alto. Neste problema, deve-se calcular a altura \(h(t)\) (em metros) desta bola em um determinado tempo \(t\) (em segundos). Considerando que a pessoa que lançou a bola, lançou \(h(t)=v_0t - \frac{gt^2}{2}\) em que \(v_0\) é a velocidade inicial da bola em metros por segundo, \(t\) é o tempo em segundos e \(g\) é a gravidade.

Assim, você deverá terminar de implementar a função alturaBola no arquivo exercicios.js substituindo os null apropriadamente. Essa função recebe como parametro o tempo \(t\), a gravidade \(g\)e a velocidade inicial \(v_0\) e escreverá na saída abaixo (usando a função escreva) a altura desta bola. Além disso, esta função deve retornar a altura dessa bola.

Para testar, invoque esta função usando a gravidade da Terra (9,81 m/s² ) velocidade inicial 50 e tempo 1, 5 e 10 (ou seja, invoque a função 3 vezes).

Lembre-se de colocar o número correspondente do exercício ao invocar a função escreva. Além disso, como já dito, para verificar se o seu código de algum erro de sintaxe (escrita), pressione F12 o erro aparecerá na aba de console.

em que, a matriz amizades cada linha e coluna correspondem a pessoas que podem manter uma relação de amizade entre si. Raiz Quadrada

Para calcular a raiz quadrada de um número, você deve usar a função Math.sqrt(numero). Por exemplo: 

let numero = 36;
let resultado = Math.sqrt(numero);
console.log(resultado); // 6

Criar uma variável

Para criar uma variável, você deve usar a palavra-chave let seguida do seu nome: 

let nomeDaVariavel;

É bem comum já declarar a variável e atribuir um valor a ela: 

let nome = 'Dorvalino';
let pontos = 53;
let estaCorrendo = true;
let armas = ['espada', 'chinelo'];
